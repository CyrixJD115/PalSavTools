功能概述：浏览和管理所有公会所有基地的物品库存和工作伙伴。在两个子选项卡之间切换：Inventory（查看/编辑容器中的物品）和Base Pals（查看/管理分配给所选基地的工作伙伴）.

基本操作（左键单击和按钮）：单击Guild按钮打开可搜索的弹出窗口并选择公会，然后单击Base按钮选择基地。单击容器列表（左面板）中的容器可在网格中查看其项目。单击 Inventory 或 Base Pals 选项卡按钮可切换视图。使用Item按钮打开跨公会物品操作（跨所有公会查找/删除物品）。使用Structure按钮进行跨公会结构删除。 Find Bases 按钮扫描所有基地以查找与项目过滤器匹配的容器。当基地有超过 30 个工作伙伴时，使用基地伙伴网格上的滚轮导航页面。

高级快捷键（双击交互）：双击库存网格中的项目以立即将其从插槽中删除（无需确认）。双击空槽以打开该槽的添加项目对话框。双击Base Pals中的某个Pal槽位：如果该槽位有Pal，则直接删除（无需确认）；如果为空，则会打开 Add New Pal 对话框。双击项目对话框搜索列表中的好友即可查找装有该项目的容器。

隐藏菜单（右键单击上下文）： 右键单击列表中的 容器 用于：添加项目（调试）、清除容器、修改容器槽位、删除容器。右键单击网格 中的 项目：编辑数量、删除项目、清除容器。右键单击 空网格槽 进行：添加项目、清除容器。右键单击 Pal 插槽，其中包含以下数据：Boss/Lucky/Awakened/DNA 切换、锁定级别（4 级）、最大所有统计数据、学习所有动作、学习技能、批量同步 Pal、删除。右键单击空的 Pal 插槽：添加新 Pal.

高级用户提示： 将鼠标悬停在 Base Pals 中的 Pal 插槽上，可在信息面板中预览其统计信息。底部的分页显示工作伙伴的当前页面 - 使用上一页/下一页按钮或滚轮进行导航。
