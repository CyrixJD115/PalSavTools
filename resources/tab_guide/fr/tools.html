Aperçu fonctionnel

  • Hub central pour la gestion des fichiers de sauvegarde : convertissez entre les formats JSON et SAV, GamePass et Steam, transférez des personnages entre les sauvegardes, corrigez les sauvegardes hôtes corrompues et injectez des emplacements de joueur.
  • Chaque outil est présenté sous forme de carte cliquable avec une icône et une description.

Opérations de base (clic gauche et boutons)

  • Cliquez sur n'importe quelle carte outil pour lancer son opération (Convertir les sauvegardes, Convertir GamePass → Steam, Convertir SteamID, Restaurer la carte, Slot Injector, Modifier la sauvegarde, Transfert de personnage, Corriger la sauvegarde de l'hôte).
  • Utilisez le bouton Load Save dans l’en-tête ou le chemin Menu → File → Load Save.
  • L'étiquette du chemin de sauvegarde sous le titre ouvre l'Explorateur de fichiers à l'emplacement de sauvegarde actuel.
  • Les icônes de navigation rapide dans la barre de statistiques (Joueurs, Guildes, Bases, Copains) accèdent directement à ces onglets.

Raccourcis avancés (interactions par double-clic)

  • Il n'y a pas de gestion du double-clic sur les cartes outils — le simple clic est le seul déclencheur.
  • Dans la boîte de dialogue Options de conversion (SAV↔JSON), le double-clic sur les éléments de la liste n'est pas pris en charge : utilisez plutôt les boutons d'option.

Menus cachés (contexte du clic droit)

  • Les cartes d'outils n'ont pas d'actions par clic droit — toutes les fonctionnalités sont uniquement accessibles par clic gauche.
  • Cependant, le bouton Menu (en haut à gauche) contient de nombreuses fonctions de nettoyage non disponibles en tant que cartes : supprimer les guildes vides, supprimer les bases/joueurs inactifs, supprimer les joueurs en double, réparer les amis illégaux, supprimer les objets/structures/amis invalides, réinitialiser les donjons/plate-forme pétrolière/envahisseur/approvisionnement, réparer toutes les structures, débloquer des coffres privés, et plus encore.

Conseil d'utilisateur expérimenté

Faites glisser et déposez un fichier .sav sur la fenêtre pour le charger instantanément - une superposition de zone de dépôt verte apparaît pour confirmer. Sauvegardez toujours avant les outils de conversion. Enregistrez régulièrement les modifications via Menu → Enregistrer les modifications.
