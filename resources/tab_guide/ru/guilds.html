Функциональный обзор

Двухпанельный вид: на верхней панели перечислены все гильдии со столбцами для названия гильдии, идентификатора гильдии и уровня гильдии. На нижней панели отображаются члены выбранной гильдии со столбцами «Имя участника», «Последнее посещение», «Уровень», «Друзья» и UID.

Основные операции (щелкните левой кнопкой мыши и кнопки)

Введите строку поиска, чтобы отфильтровать гильдии по любому столбцу. Щелкните строку гильдии, чтобы выбрать ее — нижняя панель автоматически заполняется игроками-членами этой гильдии. Нажмите на строку участника, чтобы просмотреть его подробную информацию.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши)

Двойной щелчок по строке гильдии или участника выдает сигнал item_double_clicked. Обработчик по умолчанию не подключен — двойной щелчок в настоящее время не оказывает видимого эффекта ни на одну из панелей.

Скрытые меню (контекст щелчка правой кнопкой мыши)

Щелкните правой кнопкой мыши строку гильдии, чтобы: добавить/удалить исключение, удалить гильдию, переименовать гильдию, установить уровень гильдии, разблокировать все лабораторные исследования, экспортировать базы для гильдии, импортировать базы (многофайловые), переместить выбранного игрока в выбранную гильдию. Щелкните правой кнопкой мыши строку участника , чтобы: сделать лидером гильдии, разблокировать все лабораторные исследования для гильдии, добавить/удалить исключение игрока, удалить игрока, переименовать игрока, сбросить временную метку, установить уровень игрока.

Совет опытного пользователя

Используйте Меню → Функции → Удалить пустые гильдии, чтобы удалить гильдии с нулевым количеством участников одним щелчком мыши. Идентификатор гильдии (индекс столбца 1) — это ключевая ссылка, используемая списками исключений и операциями удаления.
