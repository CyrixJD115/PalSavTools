Aperçu fonctionnel : Trois panneaux de recherche côte à côte affichant les UID des joueurs, les ID de guilde et les ID de base exclus. Les éléments répertoriés ici sont protégés contre les opérations de suppression et de nettoyage.

Opérations de base (clic gauche et boutons) : Cliquez sur n'importe quelle entrée d'exclusion pour la sélectionner. Tapez dans la barre de recherche pour filtrer les ID d’exclusion en temps réel. Ajoutez des entrées via les menus contextuels du clic droit dans les onglets Joueurs, Guildes ou Bases. Enregistrez et chargez les listes d'exclusions via Menu → Exclusions → Enregistrer les exclusions et Charger les exclusions.

Raccourcis avancés (interactions par double-clic) : Un double-clic sur une ligne d'exclusion émet le signal item_double_clicked. Aucun gestionnaire par défaut n'est connecté — le double-clic n'a actuellement aucun effet visible sur aucun des trois panneaux d'exclusion.

Menus cachés (contexte du clic droit) : Cliquez avec le bouton droit sur n'importe quelle entrée d'exclusion pour : Supprimer l'exclusion — supprime l'ID sélectionné de la liste de protection, le rendant à nouveau éligible aux opérations de nettoyage.

Conseil d'utilisateur expérimenté : Créez votre liste d'exclusion avant en exécutant des fonctions de nettoyage en masse. Des exclusions peuvent être ajoutées à partir des menus contextuels des onglets Joueurs, Guildes et Bases — vous n'avez pas besoin de saisir manuellement les identifiants ici. Enregistrez fréquemment votre liste d'exclusions via le bouton Menu.
