Descripción funcional: Tabla con capacidad de búsqueda de todos los campamentos base guardados. Las columnas muestran ID de base, ID de gremio, nombre de gremio y nivel de gremio. El menú contextual del botón derecho proporciona administración completa de la base: eliminación, ajuste de radio, importación/exportación, clonación y operaciones de gremio.

Operaciones básicas (clic izquierdo y botones): Escriba en la barra de búsqueda para filtrar bases por cualquier columna. Haga clic en una fila base para seleccionarla.

Atajos avanzados (interacciones de doble clic): Al hacer doble clic en una fila base se emite la señal item_double_clicked. No hay ningún controlador predeterminado conectado; actualmente, hacer doble clic no tiene ningún efecto visible.

Menús ocultos (contexto de clic derecho): Haga clic derecho en una fila de base para: Agregar o quitar exclusión, Eliminar base (requiere ID de base e ID de gremio), Cambiar nombre de gremio, Establecer nivel de gremio, Exportar base, Ajustar radio, Importar base, Clonar base. Tenga en cuenta que algunas acciones operan en el gremio asociado con la base en lugar de en la base misma.

Consejo de usuario avanzado: Tanto el ID base (índice de columna 0) como el ID del gremio (índice de columna 1) son necesarios para eliminarlos; el menú contextual pasa por ambos. Utilice la ruta Menú → Funciones para una limpieza masiva de bases como Eliminar bases inactivas y Eliminar objetos de mapa que no son de base.
