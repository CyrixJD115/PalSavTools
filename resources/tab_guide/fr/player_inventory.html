Aperçu fonctionnel : Gestion complète de l'inventaire du joueur avec trois sous-onglets : Inventaire (tous les objets/équipements dans le sac principal), Objets clés (objets de quête, effigies, technologie) et Stats (attributs du joueur : niveau, HP, endurance, attaque, défense, vitesse de travail, poids). Un panneau d'équipement sur la droite montre les emplacements d'armes, d'accessoires, de nourriture, d'armure, de bouclier, de planeur et de modules.

Opérations de base (clic gauche et boutons) : Cliquez sur Sélectionner un joueur... pour choisir un joueur dans la fenêtre contextuelle de recherche (tapez pour filtrer, Entrée pour sélectionner). Cliquez sur les sous-onglets pour changer de vue. Dans le sous-onglet Statistiques, cliquez sur les boutons +/− à côté de chaque statistique pour ajuster par 1, ou saisissez une valeur directement dans le champ numérique et appuyez sur Entrée. Cliquez sur les boutons Level +/− ou saisissez un niveau (1 à 80). Cliquez sur le bouton Déverrouiller toutes les cartes + Voyage rapide dans l'onglet Inventaire pour révéler tous les points de voyage rapide. Cliquez sur les emplacements d'équipement verrouillés (arme/armure/nourriture/accessoire) pour tenter de les déverrouiller : une info-bulle vous indique quel objet est nécessaire.

Raccourcis avancés (interactions par double-clic) : Double-cliquez sur n'importe quel élément de la grille Inventaire ou Éléments clés pour le supprimer directement sans invite de confirmation. Double-cliquez sur un emplacement vide pour ouvrir la boîte de dialogue d'ajout d'élément. Double-cliquez sur un emplacement d'équipement contenant un élément pour supprimer cet élément ; double-cliquez sur un emplacement d'équipement vide pour ouvrir le sélecteur d'éléments supplémentaires pour cet emplacement.

Menus cachés (contexte du clic droit) : Cliquez avec le bouton droit sur un emplacement d'inventaire rempli pour : modifier la quantité, supprimer, ajouter un article. Faites un clic droit sur un emplacement vide pour : Ajouter un élément, Effacer un emplacement (utile pour les emplacements corrompus). Cliquez avec le bouton droit sur un emplacement d'équipement pour : Modifier la quantité (emplacements de nourriture uniquement), Effacer l'emplacement ou Ajouter un article. La boîte de dialogue de sélection d'éléments prend également en charge le double-clic sur un résultat de recherche pour ajouter immédiatement l'élément.

Conseil d'utilisateur expérimenté : Passez la souris sur n'importe quel élément pour voir une info-bulle avec le nom de l'élément, la quantité, l'ID interne et la description complète du jeu. Le bouton Trier organise les éléments par catégorie. L'onglet Éléments clés comporte des boutons « Ajouter toutes les effigies » et « Ajouter tous les éléments clés » dédiés aux opérations groupées.
