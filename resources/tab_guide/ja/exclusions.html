機能概要: 除外されたプレイヤー UID、ギルド ID、およびベース ID を表示する 3 つの横並びの検索パネル。ここにリストされている項目は、削除およびクリーンアップ操作から保護されています。

基本操作 (左クリックとボタン): 除外エントリをクリックして選択します。検索バーに入力して除外 ID をリアルタイムでフィルタリングします。 [プレイヤー]、[ギルド]、または [拠点] タブで右クリックしてコンテキスト メニューを使用してエントリを追加します。 Menu → Exclusions → Save Exclusions および Load Exclusions.

を使用して除外リストを保存およびロードします。

高度なショートカット (ダブルクリック操作): 除外行をダブルクリックすると、item_double_clicked シグナルが生成されます。デフォルトのハンドラーが接続されていません。現在、ダブルクリックしても 3 つの除外パネルのいずれにも目に見える影響はありません。

隠しメニュー (コンテキストを右クリック): 次の除外エントリを右クリックします: 除外の削除 — 選択した ID を保護リストから削除し、再びクリーンアップ操作の対象になります。

パワー ユーザー ヒント: 一括クリーンアップ関数を実行する 前 に除外リストを作成します。除外は、[プレイヤー]、[ギルド]、および [拠点] タブの右クリック メニューから追加できます。ここで ID を手動で入力する必要はありません。 [メニュー] ボタンを使用して、除外リストを頻繁に保存します。
