기능 개요

세이브에 있는 모든 베이스캠프의 검색 가능한 테이블입니다. 열에는 기본 ID, 길드 ID, 길드 이름, 길드 레벨이 표시됩니다. 마우스 오른쪽 버튼 클릭 컨텍스트 메뉴는 삭제, 반경 조정, 가져오기/내보내기, 복제, 길드 운영 등 전체 기지 관리를 제공합니다.

기본 작업(왼쪽 클릭 및 버튼)

검색창에 입력하여 특정 열을 기준으로 베이스를 필터링하세요. 기본 행을 클릭하여 선택하세요.

고급 바로 가기(두 번 클릭 상호 작용)

기본 행을 두 번 클릭하면 item_double_clicked 신호가 발생합니다. 기본 핸들러가 연결되어 있지 않습니다. 현재 두 번 클릭해도 눈에 보이는 효과가 없습니다.

숨겨진 메뉴(컨텍스트 마우스 오른쪽 버튼 클릭)

제외 추가/제거, 베이스 삭제(베이스 ID 및 길드 ID 필요), 길드 이름 변경, 길드 레벨 설정, 베이스 내보내기, 반경 조정, 베이스 가져오기, 베이스 복제를 위해 베이스 행을 마우스 오른쪽 버튼으로 클릭합니다. 일부 작업은 기지 자체가 아닌 기지와 연결된 길드에서 작동한다는 점에 유의하세요.

파워 유저 팁

삭제하려면 기본 ID(열 인덱스 0)와 길드 ID(열 인덱스 1)가 모두 필요합니다. 컨텍스트 메뉴는 둘 다 전달합니다. 비활성 베이스 삭제 및 비베이스 맵 객체 삭제와 같은 대량 베이스 정리를 위해서는 메뉴 → 함수 경로를 사용하세요.
