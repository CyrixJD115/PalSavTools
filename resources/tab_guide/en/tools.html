Functional Overview: Central hub for save file management — convert between JSON and SAV, GamePass and Steam formats, transfer characters between saves, fix corrupted host saves, and inject player slots. Each tool is presented as a clickable card with an icon and description.

Basic Operations (Left-Click & Buttons): Click any tool card to launch its operation (Convert Saves, Convert GamePass→Steam, Convert SteamID, Restore Map, Slot Injector, Modify Save, Character Transfer, Fix Host Save). Use the Load Save button in the header or the Menu → File → Load Save path. The save path label below the title opens File Explorer to the current save location. Quick-nav icons in the stats bar (Players, Guilds, Bases, Pals) jump directly to those tabs.

Advanced Shortcuts (Double-Click Interactions): There is no double-click handling on tool cards — single-click is the only trigger. In the Conversion Options dialog (SAV↔JSON), double-click on list items is not supported — use the option buttons instead.

Hidden Menus (Right-Click Context): Tool cards have no right-click actions — all features are left-click only. However, the Menu button (top-left) contains many cleanup Functions not available as cards: Delete Empty Guilds, Delete Inactive Bases/Players, Delete Duplicate Players, Fix Illegal Pals, Remove Invalid Items/Structures/Pals, Reset Dungeons/Oil Rig/Invader/Supply, Repair All Structures, Unlock Private Chests, and more.

Power-User Tip: Drag-and-drop a .sav file onto the window to load it instantly — a green drop zone overlay appears to confirm. Always back up before conversion tools. Save changes regularly via Menu → Save Changes.
