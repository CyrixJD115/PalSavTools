Обзор функций

Интерфейс глубокого редактирования для любого приятеля, принадлежащего любому игроку. Просматривайте друзей, организованных по группам (активный отряд) и Palbox (хранилище), с полным контролем над статистикой, IV, активными навыками, пассивными чертами, пригодностью для работы, рангом, улучшениями души, флагами внешнего вида (Босс/Счастливчик/Пробужденный/ДНК) и многим другим. Информационная панель справа показывает все редактируемые детали.

Основные операции (щелкните левой кнопкой мыши и кнопки)

Нажмите Выбрать игрока..., чтобы выбрать игрока из всплывающего окна с возможностью поиска (введите для фильтрации, Enter для выбора). Нажмите на слот Party или Palbox, чтобы выбрать приятеля для редактирования — информационная панель заполнится всеми редактируемыми полями. Нажимайте кнопки переключения в заголовке информационной панели: Пол, Альфа-босс, Удачливый/Блестящий, Пробужденный, Импортированный/ДНК, Максимальные все характеристики. Нажмите звездочку «Избранное», чтобы переключаться между уровнями блокировки (0–3). Щелкните поля меток, чтобы редактировать их в диалоговом окне ввода: имя (псевдоним), уровень (1–80), рейтинг доверия (0–10). Щелкните метки IV (HP/Атака/Защита), чтобы установить значение от 0 до 100. Нажмите на метки души (HP/Атака/Защита/Скорость создания), чтобы установить значение от 0 до 20. Нажмите на активные ячейки навыков, чтобы открыть окно выбора навыков. Нажмите на пассивные слоты, чтобы открыть окно выбора пассивов. Нажмите на значения пригодности к работе, чтобы установить их уровень (0–10). Перемещайтесь между страницами Palbox с помощью кнопок ← Предыдущий / Следующий → или колесика мыши по сетке.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши)

Дважды щелкните слот Party или Palbox, в котором есть приятель, чтобы быстро удалить его (нет подтверждения — вызывает то же действие «delete_direct», что и в контекстном меню). Дважды щелкните пустой слот Party или Palbox, чтобы открыть диалоговое окно создания «Добавить нового приятеля». Дважды щелкните приятеля в списке «Добавить нового приятеля», чтобы немедленно создать его. Дважды щелкните строку заголовка FramelessDialog, чтобы переключить развертывание.

Скрытые меню (контекст правой кнопкой мыши)

Щелкните правой кнопкой мыши любой слот группы или Palbox с приятелем для: переключения босса «Альфа», переключения «Счастливчик» / «Блестящий», переключения «Пробуждение», переключения «Импортировать/ДНК», «Уровень блокировки» (подменю: 4 уровня), «Максимум всех характеристик», «Изучить все приемы», «Диалог изученных навыков», «Массовая синхронизация приятеля», «Удалить». Щелкните правой кнопкой мыши пустой слот для: «Добавить нового приятеля». Щелкните правой кнопкой мыши портрет приятеля на информационной панели, чтобы открыть то же расширенное контекстное меню.

Совет для опытных пользователей

Сочетания клавиш ускоряют редактирование: Q = предыдущая страница Palbox, E = следующая страница Palbox, F = фокус и прокрутка до информационной панели, C = переключение просмотра навыков. Наведите указатель мыши на слот, чтобы просмотреть его Pal на информационной панели, не выбирая его. Используйте Массовую синхронизацию (из контекстного меню), чтобы копировать статистику/навыки/пассивные умения с одного приятеля на несколько других одновременно.
