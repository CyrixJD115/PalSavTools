Функциональный обзор: Три расположенные рядом панели поиска, отображающие исключенные UID игроков, идентификаторы гильдий и базовые идентификаторы. Перечисленные здесь элементы защищены от операций удаления и очистки.

Основные операции (щелкните левой кнопкой мыши и кнопки): Щелкните любую запись исключения, чтобы выбрать ее. Введите строку поиска, чтобы фильтровать идентификаторы исключений в режиме реального времени. Добавляйте записи через контекстное меню, вызываемое правой кнопкой мыши на вкладках «Игроки», «Гильдии» или «Базы». Сохраните и загрузите списки исключений через Меню → Исключения → Сохранить исключения и Загрузить исключения.

.

Расширенные ярлыки (взаимодействие при двойном щелчке): Двойной щелчок по строке исключения выдает сигнал item_double_clicked. Обработчик по умолчанию не подключен — двойной щелчок в настоящее время не оказывает видимого влияния ни на одну из трех панелей исключений.

Скрытые меню (контекст, вызываемый правой кнопкой мыши): Щелкните правой кнопкой мыши любую запись исключения для: Удалить исключение — удаляет выбранный идентификатор из списка защиты, делая его снова доступным для операций очистки.

Совет для опытных пользователей: Создайте список исключений перед запуском любых функций массовой очистки. Исключения можно добавлять из контекстного меню на вкладках «Игроки», «Гильдии» и «Базы» — здесь не нужно вручную вводить идентификаторы. Чаще сохраняйте список исключений с помощью кнопки «Меню».
