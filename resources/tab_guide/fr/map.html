Aperçu fonctionnel : Carte du monde interactive montrant les marqueurs de base (icône de maison), les marqueurs de joueur (icône de personne) et les zones d'exclusion. Cliquez sur les marqueurs pour inspecter les bases/joueurs, cliquez avec le bouton droit pour les actions de gestion et utilisez l'arborescence de la barre latérale pour naviguer par guilde.

Opérations de base (clic gauche et boutons) : Cliquez sur un marqueur pour le sélectionner (surbrillance avec effet lumineux). Faites défiler la molette de la souris pour zoomer (1,15x par étape, plage de 1,0x à 30,0x). Cliquez et faites glisser l'espace vide pour effectuer un panoramique. Basculez les superpositions de cartes via la barre d'outils : boutons Bases, Joueurs, Anneaux de rayon et Zones d'exclusion. Basculez entre les vues Carte du monde et Carte arborescente. La barre de recherche filtre les marqueurs par guilde ou nom de joueur. Les boutons de l'onglet Bases/Joueurs sur la gauche permettent de basculer entre les panneaux de l'arborescence de base et de l'arborescence des joueurs. Cliquez sur Calibrate pour entrer en mode d'étalonnage pour un alignement précis des coordonnées.

Raccourcis avancés (interactions par double-clic) : Double-cliquez sur n'importe quel marqueur de base ou de joueur pour zoomer et animer la vue de la carte sur ce marqueur avec un zoom 26,0x tout en mettant à jour le panneau d'informations avec ses détails. Double-cliquez sur un élément de zone (zone d'exclusion dessinée sur la carte) pour ouvrir une boîte de dialogue de renommage. En mode dessin de zone avec Rect sélectionné : un premier double-clic définit le coin A, un deuxième double-clic définit le coin B et crée la zone. En mode Polygone avec au moins 3 points : un double-clic ferme et crée la zone polygone.

Menus cachés (contexte du clic droit) : Cliquez avec le bouton droit sur un marqueur de base pour : Supprimer la base, Exporter la base, Ajuster le rayon. Cliquez avec le bouton droit sur un marqueur de joueur pour : Supprimer un joueur, Renommer un joueur, Déverrouiller la cage de visualisation, Déverrouiller des technologies. Cliquez avec le bouton droit sur espace de carte vide pour : importer la base, dessiner des zones, effacer toutes les zones, exporter des zones, importer des zones. Cliquez avec le bouton droit sur les éléments de l'arborescence base/joueur pour des actions supplémentaires, notamment des opérations au niveau de la guilde (Renommer la guilde, supprimer la guilde, exporter/importer les bases de la guilde, mettre à jour les identifiants des conteneurs). Cliquez avec le bouton droit sur une zone sur la carte pour : Supprimer la zone (avec confirmation), Renommer la zone.

Conseil d'utilisateur expérimenté : Appuyez sur Escape lorsque vous êtes en mode de dessin de zone pour annuler et quitter immédiatement. Utilisez le mode de calibrage (bouton Calibrer) pour aligner précisément la carte avec les coordonnées du jeu en cliquant avec le bouton gauche sur les emplacements connus et en cliquant avec le bouton droit pour annuler.
