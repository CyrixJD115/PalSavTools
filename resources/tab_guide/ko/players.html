기능 개요: 저장에 있는 모든 플레이어의 검색 가능한 테이블입니다. 열에는 플레이어 이름, 마지막 접속, 레벨, 친구 수, UID, 길드 이름, 길드 ID 및 길드 레벨이 표시됩니다. 여러 플레이어의 아이템, Pal 및 기술 관리를 위해 하단의 일괄 작업 버튼을 사용하세요.

기본 작업(왼쪽 클릭 및 버튼): 실시간으로 플레이어를 필터링하려면 검색창에 입력하세요. 모든 열 일치 항목이 표시됩니다. 행을 클릭하여 선택합니다. 대량 항목 관리을 클릭하여 플레이어 항목 작업 대화 상자를 엽니다(선택한 플레이어 전체에 항목 추가/제거). Pal 작업을 하려면 대량 Pal 관리을 클릭하세요. 기술 잠금을 해제하려면 대량 기술 관리를 클릭하세요.

고급 단축키(더블 클릭 상호 작용): 플레이어 행을 두 번 클릭하면 item_double_clicked 신호가 발생합니다. 그러나 현재 코드에는 기본 핸들러가 연결되어 있지 않습니다. 현재 두 번 클릭해도 눈에 띄는 효과가 없습니다. (향후 사용 예정입니다.)

숨겨진 메뉴(컨텍스트 마우스 오른쪽 버튼 클릭): 제외 추가/제거, 플레이어 삭제, 플레이어 이름 바꾸기, 보기 케이지 잠금 해제, 타임스탬프 재설정, 컨테이너 ID 업데이트, 모든 기술 잠금 해제, 기술 포인트 편집, 플레이어 통계 편집, 플레이어 레벨 설정을 위해 플레이어 행을 마우스 오른쪽 버튼으로 클릭합니다. 플레이어의 길드에 대한 길드 수준 작업도 포함됩니다: 길드 리더 만들기, 길드 삭제, 길드 이름 바꾸기, 모든 연구실 연구 잠금 해제, 길드 수준 설정, 길드로 기지 가져오기.

고급 사용자 팁: 헤더를 클릭하여 열을 정렬하세요. UID 열(열 인덱스 4)은 제외 및 삭제 작업의 키로 사용됩니다. 버려진 플레이어를 일괄 정리하려면 마지막 접속 날짜를 확인한 후 메뉴 → 기능 → 비활성 플레이어 삭제를 사용하세요.
