Функциональный обзор: Просматривайте и управляйте инвентарем предметов и рабочими приятелями для всех баз всех гильдий. Переключение между двумя подвкладками: Inventory (просмотр/редактирование предметов в контейнерах) и Base Pals (просмотр/управление рабочими Pals, закрепленными за выбранной базой).

Основные операции (щелкните левой кнопкой мыши и кнопки): Нажмите кнопку Guild, чтобы открыть всплывающее окно с возможностью поиска и выберите гильдию, затем нажмите кнопку Base, чтобы выбрать базу. Щелкните контейнер в списке контейнеров (левая панель), чтобы просмотреть его элементы в сетке. Нажмите кнопки вкладок Inventory или Base Pals, чтобы переключить представления. Используйте кнопку Item, чтобы открыть операции с предметами между гильдиями (найти/удалить предметы во всех гильдиях). Используйте кнопку Structure для удаления структуры между гильдиями. Кнопка Найти базы сканирует все базы на наличие контейнеров, соответствующих фильтру предметов. Используйте колесо прокрутки в сетке «Базовые друзья» для навигации по страницам, когда в базе более 30 рабочих друзей.

Расширенные ярлыки (взаимодействие с двойным щелчком): Дважды щелкните предмет в сетке инвентаря, чтобы немедленно удалить его из слота (без подтверждения). Дважды щелкните пустой слот, чтобы открыть диалоговое окно добавления элемента для этого слота. Дважды щелкните слот Pal в Base Pals: если в слоте есть Pal, он удаляется напрямую (без подтверждения); если оно пусто, откроется диалоговое окно Добавить нового приятеля. Дважды щелкните приятеля в списке поиска диалоговых окон с предметами, чтобы найти контейнеры, содержащие этот предмет.

Скрытые меню (контекст, вызываемый правой кнопкой мыши): Щелкните правой кнопкой мыши контейнер в списке, чтобы выполнить следующие действия: «Добавить элемент (отладка), «Очистить контейнер», «Изменить слоты контейнера», «Удалить контейнер». Щелкните правой кнопкой мыши элемент в сетке, чтобы: «Изменить количество», «Удалить элемент», «Очистить контейнер». Щелкните правой кнопкой мыши пустой слот сетки , чтобы: «Добавить элемент», «Очистить контейнер». Щелкните правой кнопкой мыши слот Pal с данными для: переключателей «Босс/Удачливый/Пробужденный/ДНК», «Уровень блокировки» (4 уровня), «Максимальные все характеристики», «Изучение всех приемов», «Изученные навыки», «Массовая синхронизация приятеля», «Удалить». Щелкните правой кнопкой мыши пустой слот Pal, чтобы: Добавить нового приятеля.

.

Совет для опытных пользователей: Наведите указатель мыши на слот Pal в Base Pals, чтобы просмотреть его статистику на информационной панели. Пагинация внизу показывает текущую страницу рабочего Pals — для навигации используйте кнопки «Предыдущая/Следующая» или колесо прокрутки.
