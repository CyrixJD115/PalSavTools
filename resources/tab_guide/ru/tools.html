Обзор функций: Центральный центр управления файлами сохранений — преобразование между форматами JSON и SAV, GamePass и Steam, перенос персонажей между сохранениями, исправление поврежденных сохранений хоста и добавление слотов игроков. Каждый инструмент представлен в виде кликабельной карточки со значком и описанием.

Основные операции (щелкните левой кнопкой мыши и кнопки): Щелкните любую карточку инструментов, чтобы запустить ее работу (конвертировать сохранения, конвертировать GamePass→Steam, конвертировать SteamID, восстановить карту, инжектор слотов, изменить сохранение, перенос персонажа, исправить сохранение хоста). Используйте кнопку Загрузить сохранение в заголовке или путь Меню → Файл → Загрузить сохранение. Метка пути сохранения под заголовком открывает проводник к текущему месту сохранения. Значки быстрой навигации на панели статистики (Игроки, Гильдии, Базы, Друзья) переходят непосредственно на эти вкладки.

Расширенные сочетания клавиш (взаимодействие с двойным щелчком мыши): На карточках инструментов обработка двойного щелчка не предусмотрена — единственным триггером является одиночный щелчок. В диалоговом окне «Параметры преобразования» (SAV↔JSON) двойной щелчок по элементам списка не поддерживается — вместо этого используйте кнопки выбора.

Скрытые меню (контекст, вызываемый правой кнопкой мыши): На картах инструментов нет действий, вызываемых щелчком правой кнопкой мыши — все функции доступны только щелчком левой кнопкой мыши. Однако кнопка Menu (вверху слева) содержит множество функций очистки, недоступных в виде карточек: удаление пустых гильдий, удаление неактивных баз/игроков, удаление дубликатов игроков, исправление нелегальных друзей, удаление недействительных предметов/структур/приятелей, сброс подземелий/нефтяной вышки/захватчика/припасов, ремонт всех построек, разблокировка частных сундуков и подробнее.

Совет для опытных пользователей: Перетащите файл .sav в окно, чтобы мгновенно загрузить его — для подтверждения появится зеленая зона перетаскивания. Всегда делайте резервную копию перед использованием инструментов преобразования. Регулярно сохраняйте изменения через Меню → Сохранить изменения.
