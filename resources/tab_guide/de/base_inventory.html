Funktionsübersicht

  • Durchsuchen und verwalten Sie Gegenstandsbestände und Arbeitsfreunde für alle Basen in allen Gilden.
  • Wechseln Sie zwischen zwei Unterregisterkarten: Inventory (Artikel in Containern anzeigen/bearbeiten) und Base Pals (Arbeitspals anzeigen/verwalten, die der ausgewählten Basis zugewiesen sind).

Grundfunktionen (Linksklick und Schaltflächen)

  • Klicken Sie auf die Schaltfläche Gilde, um ein durchsuchbares Popup zu öffnen und eine Gilde auszuwählen.
  • Klicken Sie dann auf die Schaltfläche Basis, um eine Basis auszuwählen.
  • Klicken Sie auf einen Container in der Containerliste (linker Bereich), um seine Elemente im Raster anzuzeigen.
  • Klicken Sie auf die Registerkartenschaltflächen Inventory oder Base Pals, um die Ansicht zu wechseln.
  • Verwenden Sie die Schaltfläche Item, um gildenübergreifende Gegenstandsoperationen zu öffnen (Gegenstände in allen Gilden finden/entfernen).
  • Verwenden Sie die Schaltfläche Structure, um eine gildenübergreifende Struktur zu löschen.
  • Die Schaltfläche Find Bases durchsucht alle Basen nach Containern, die einem Artikelfilter entsprechen.
  • Verwenden Sie das Scrollrad im Base Pals-Raster, um durch die Seiten zu navigieren, wenn eine Basis mehr als 30 aktive Pals hat.

Erweiterte Verknüpfungen (Doppelklick-Interaktionen)

  • Doppelklicken Sie auf einen Gegenstand im Inventarraster, um ihn sofort aus dem Steckplatz zu entfernen (keine Bestätigung).
  • Doppelklicken Sie auf einen leeren Steckplatz, um das Dialogfeld zum Hinzufügen von Elementen für diesen Steckplatz zu öffnen.
  • Doppelklicken Sie auf einen Pal-Slot in Base Pals: Wenn der Slot einen Pal hat, wird er direkt gelöscht (keine Bestätigung); Wenn es leer ist, wird das Dialogfeld „Neuen Pal hinzufügen“ geöffnet.
  • Doppelklicken Sie in der Suchliste der Artikeldialoge auf einen Kumpel, um Container zu finden, die diesen Artikel enthalten.

Versteckte Menüs (Rechtsklick-Kontext)

  • Klicken Sie mit der rechten Maustaste auf einen Container in der Liste für: Element hinzufügen (Debug), Container löschen, Container-Slots ändern, Container löschen.
  • Klicken Sie mit der rechten Maustaste auf einen Artikel im Raster für: Menge bearbeiten, Artikel entfernen, Behälter leeren.
  • Klicken Sie mit der rechten Maustaste auf einen leeren Rasterplatz für: Element hinzufügen, Container löschen.
  • Klicken Sie mit der rechten Maustaste auf einen Pal-Slot mit Daten für: Boss/Lucky/Awakened/DNA-Schalter, Sperrstufe (4 Stufen), Max.
  • Alle Statistiken, Alle Bewegungen lernen, Erlernte Fertigkeiten, Bulk-Sync-Pal, Löschen.
  • Klicken Sie mit der rechten Maustaste auf einen leeren Pal-Slot für: Neuen Pal hinzufügen.

Power-User-Tipp

Bewegen Sie den Mauszeiger über einen Pal-Slot in Base Pals, um eine Vorschau seiner Statistiken im Infofeld anzuzeigen. Die Paginierung unten zeigt die aktuelle Seite der aktiven Pals – verwenden Sie die Schaltflächen „Zurück/Weiter“ oder das Scrollrad, um zu navigieren.
