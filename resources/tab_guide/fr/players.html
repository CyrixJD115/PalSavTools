Aperçu fonctionnel : Tableau consultable de tous les joueurs de la sauvegarde. Les colonnes indiquent le nom du joueur, la dernière vue, le niveau, le nombre de copains, l'UID, le nom de la guilde, l'identifiant de la guilde et le niveau de la guilde. Utilisez les boutons d'action groupée en bas pour la gestion des objets, des amis et de la technologie sur plusieurs joueurs.

Opérations de base (clic gauche et boutons) : Tapez dans la barre de recherche pour filtrer les joueurs en temps réel — toute correspondance de colonne s'affichera. Cliquez sur une ligne pour la sélectionner. Cliquez sur Gestion des éléments en masse pour ouvrir la boîte de dialogue Action des éléments du joueur (ajouter/supprimer des éléments sur les joueurs sélectionnés). Cliquez sur Bulk Pal Management pour les opérations Pal. Cliquez sur Bulk Technology Management pour les déverrouillages technologiques.

Raccourcis avancés (interactions par double-clic) : Un double-clic sur une ligne de joueur émet le signal item_double_clicked. Cependant, aucun gestionnaire par défaut n'est connecté dans le code actuel — le double-clic n'a actuellement aucun effet visible. (Destiné à une utilisation future.)

Menus cachés (contexte du clic droit) : Cliquez avec le bouton droit sur une ligne de joueur pour : ajouter/supprimer une exclusion, supprimer un joueur, renommer un joueur, déverrouiller la cage de visualisation, réinitialiser l'horodatage, mettre à jour les identifiants des conteneurs, déverrouiller toutes les technologies, modifier les points techniques, modifier les statistiques du joueur, définir le niveau du joueur. Inclut également des actions au niveau de la guilde du joueur : devenir chef de guilde, supprimer une guilde, renommer une guilde, débloquer toutes les recherches en laboratoire, définir le niveau de guilde, importer une base dans une guilde.

Conseil d'utilisateur expérimenté : Triez n'importe quelle colonne en cliquant sur son en-tête. La colonne UID (index de colonne 4) est utilisée comme clé pour les opérations d'exclusion et de suppression. Utilisez Menu → Fonctions → Supprimer les joueurs inactifs après avoir vérifié les dates de dernière visite pour nettoyer en masse les joueurs abandonnés.
