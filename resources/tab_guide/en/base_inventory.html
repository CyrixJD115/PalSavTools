Functional Overview: Browse and manage item inventories and working Pals for all bases across all guilds. Switch between two sub-tabs: Inventory (view/edit items in containers) and Base Pals (view/manage working Pals assigned to the selected base).

Basic Operations (Left-Click & Buttons): Click the Guild button to open a searchable popup and select a guild, then click the Base button to select a base. Click a container in the container list (left panel) to view its items in the grid. Click the Inventory or Base Pals tab buttons to switch views. Use the Item button to open cross-guild item operations (find/remove items across all guilds). Use the Structure button for cross-guild structure deletion. The Find Bases button scans all bases for containers matching an item filter. Use the scroll wheel on the Base Pals grid to navigate pages when a base has more than 30 working Pals.

Advanced Shortcuts (Double-Click Interactions): Double-click an item in the inventory grid to remove it from the slot immediately (no confirmation). Double-click an empty slot to open the add-item dialog for that slot. Double-click a Pal slot in Base Pals: if the slot has a Pal, it is deleted directly (no confirmation); if empty, it opens the Add New Pal dialog. Double-click a pal in the search list of item dialogs to find containers holding that item.

Hidden Menus (Right-Click Context): Right-click a container in the list for: Add Item (debug), Clear Container, Modify Container Slots, Delete Container. Right-click an item in the grid for: Edit Quantity, Remove Item, Clear Container. Right-click an empty grid slot for: Add Item, Clear Container. Right-click a Pal slot with data for: Boss/Lucky/Awakened/DNA toggles, Lock Level (4 levels), Max All Stats, Learn All Moves, Learnt Skills, Bulk Sync Pal, Delete. Right-click an empty Pal slot for: Add New Pal.

Power-User Tip: Hover over a Pal slot in Base Pals to preview its stats in the info panel. The pagination at the bottom shows the current page of working Pals — use the prev/next buttons or scroll wheel to navigate.
