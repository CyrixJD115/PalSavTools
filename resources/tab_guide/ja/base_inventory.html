機能概要: すべてのギルドのすべての拠点のアイテム インベントリと作業仲間を参照および管理します。 2 つのサブタブを切り替えます: Inventory (コンテナ内のアイテムの表示/編集) と Base Pals (選択したベースに割り当てられた作業 Pals の表示/管理)。

基本操作 (左クリックとボタン): ギルド ボタンをクリックして検索可能なポップアップを開き、ギルドを選択し、次に Base ボタンをクリックして拠点を選択します。コンテナ リスト (左側のパネル) でコンテナをクリックすると、そのアイテムがグリッドに表示されます。 [Inventory] または [Base Pals] タブ ボタンをクリックしてビューを切り替えます。 Item ボタンを使用して、ギルド間のアイテム操作 (すべてのギルドにわたるアイテムの検索/削除) を開きます。ギルド間の構造を削除するには、Structure ボタンを使用します。 Find Bases ボタンは、項目フィルターに一致するコンテナーのすべてのベースをスキャンします。拠点に 30 人を超える作業仲間がいる場合は、拠点仲間グリッドのスクロール ホイールを使用してページを移動します。

高度なショートカット (ダブルクリック インタラクション): インベントリ グリッド内のアイテムをダブルクリックすると、スロットからすぐに削除されます (確認なし)。空のスロットをダブルクリックすると、そのスロットの項目追加ダイアログが開きます。 Base Pals の Pal スロットをダブルクリックします。スロットに Pal がある場合、直接削除されます (確認なし)。空の場合は、Add New Pal ダイアログが開きます。アイテム ダイアログの検索リストで仲間をダブルクリックすると、そのアイテムが入っているコンテナが見つかります。

隠しメニュー (コンテキストを右クリック): リスト内の コンテナを右クリックします: 項目の追加 (デバッグ)、コンテナのクリア、コンテナ スロットの変更、コンテナの削除。グリッドのアイテムを右クリックして、「数量の編集」、「アイテムの削除」、「コンテナのクリア」を選択します。 空のグリッド スロットを右クリックして、項目の追加、コンテナのクリアを行います。次のデータを含むパルスロットを右クリックします: ボス/ラッキー/覚醒/DNA切り替え、ロックレベル(4レベル)、すべてのステータスの最大値、すべての動きの学習、習得したスキル、バルク同期パル、削除。空の Pal スロットを右クリックして、新しい Pal を追加します。

パワーユーザー向けヒント: Base Pals の Pal スロットにカーソルを置くと、情報パネルで統計をプレビューできます。下部のページネーションには、作業中の Pal の現在のページが表示されます。移動するには、前/次ボタンまたはスクロール ホイールを使用します。
