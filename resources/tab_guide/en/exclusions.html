Functional Overview: Three side-by-side search panels displaying excluded Player UIDs, Guild IDs, and Base IDs. Items listed here are protected from deletion and cleanup operations.

Basic Operations (Left-Click & Buttons): Click any exclusion entry to select it. Type in the search bar to filter exclusion IDs in real-time. Add entries via right-click context menus in the Players, Guilds, or Bases tabs. Save and load exclusion lists via Menu → Exclusions → Save Exclusions and Load Exclusions.

Advanced Shortcuts (Double-Click Interactions): Double-clicking an exclusion row emits the item_double_clicked signal. No default handler is connected — double-click currently has no visible effect on any of the three exclusion panels.

Hidden Menus (Right-Click Context): Right-click any exclusion entry for: Remove Exclusion — removes the selected ID from the protection list, making it eligible for cleanup operations again.

Power-User Tip: Build your exclusion list before running any bulk cleanup functions. Exclusions can be added from the right-click menus in Players, Guilds, and Bases tabs — you do not need to manually type IDs here. Save your exclusion list frequently via the Menu button.
