Функциональный обзор: Интерактивная карта мира, на которой показаны базовые маркеры (значок дома), маркеры игрока (значок человека) и запретные зоны. Нажимайте маркеры, чтобы проверять базы/игроков, щелкните правой кнопкой мыши для действий по управлению и используйте дерево боковой панели для навигации по гильдии.

Основные операции (щелкните левой кнопкой мыши и кнопки): Щелкните маркер, чтобы выбрать его (выделится эффектом свечения). Прокрутите колесо мыши для увеличения (1,15x за шаг, диапазон 1,0x–30,0x). Нажмите и перетащите пустое место для панорамирования. Переключение наложений карты с помощью панели инструментов: кнопки «Базы», ​​«Игроки», «Радиусные кольца» и «Зоны отчуждения». Переключение между представлениями «Карта мира» и «Карта дерева». Панель поиска фильтрует маркеры по имени гильдии или игрока. Кнопки вкладки «Базы/Игроки» слева переключают панели базового дерева и дерева игроков. Нажмите Calibrate, чтобы войти в режим калибровки для точного выравнивания координат.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши): Дважды щелкните любой маркер базы или игрока, чтобы увеличить масштаб и анимировать вид карты для этого маркера с увеличением 26,0x, одновременно обновляя информационную панель с ее подробностями. Дважды щелкните элемент зоны (зона отчуждения, нарисованная на карте), чтобы открыть диалоговое окно переименования. В режиме рисования зон с выбранным прямоугольником: первый двойной щелчок устанавливает угол A, второй двойной щелчок устанавливает угол B и создает зону. В режиме «Многоугольник» с минимум 3 точками: двойной щелчок закроет и создаст зону многоугольника.

Скрытые меню (контекст, вызываемый правой кнопкой мыши): Щелкните правой кнопкой мыши базовый маркер, чтобы выполнить следующие действия: «Удалить базу», «Экспортировать базу», «Отрегулировать радиус». Щелкните правой кнопкой мыши маркер player, чтобы: удалить игрока, переименовать игрока, разблокировать клетку просмотра, разблокировать технологии. Щелкните правой кнопкой мыши пустое пространство карты, чтобы: «Импортировать базу», «Нарисовать зоны», «Очистить все зоны», «Экспортировать зоны», «Импортировать зоны». Щелкните правой кнопкой мыши элементы в дереве базы/игрока, чтобы выполнить дополнительные действия, включая операции на уровне гильдии (переименовать гильдию, удалить гильдию, экспортировать/импортировать базы для гильдии, обновить идентификаторы контейнеров). Щелкните правой кнопкой мыши zone на карте, чтобы: Удалить зону (с подтверждением), Переименовать зону.

.

Совет для опытных пользователей: Нажмите Escape в режиме рисования зон, чтобы отменить операцию и немедленно выйти. Используйте режим калибровки (кнопка «Калибровать»), чтобы точно совместить карту с игровыми координатами, щелкнув левой кнопкой мыши по известным местоположениям и щелкнув правой кнопкой мыши, чтобы отменить действие.
